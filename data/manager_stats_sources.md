# Manager stats sources (2025/26)

Definition used for `major_trophies_top5`:
- Count only **managerial honours** won with clubs in the **top‑5 leagues** (England, Spain, Germany, Italy, France).
- Include **top‑flight league titles**, **primary domestic cups** (FA Cup/Copa del Rey/Coppa Italia/DFB‑Pokal/Coupe de France), **secondary domestic cups** (e.g., League Cup/EFL Cup), **UEFA Champions/Europa/Conference League**, **UEFA Super Cup**, and **FIFA Club World Cup**.
- **Exclude** Community Shield, friendlies, lower‑tier titles, and runner‑up finishes.

PPG source: SofaScore manager profiles. If a SofaScore profile is not found, the PPG is left blank.

PPG sources are listed per manager (Transfermarkt or SofaScore). Trophy sources use Wikipedia honours pages.
