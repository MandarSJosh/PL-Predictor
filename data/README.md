Data folder

This folder stores scraped and engineered datasets used by the pipeline.

Typical files:
- `raw_matches.csv`: Raw match results scraped from FBref
- `features.csv`: Engineered features used for model training

Notes:
- These files can be regenerated by running the pipeline:
  - `python -m src.pipeline --collect --features`
- For sharing with interviewers, consider excluding large data files and
  providing this README with reproduction steps instead.
